  * proxy ip 할당 방법 순환 방법

- 진행중 관리
  * profile - proxy ip 관리 < ip는 순환해서 사용
  * block 되면 profile 삭제하고 새로운 profile과 proxy ip 로 새로 브라우저 띄운다.

- 모든 페이지 https://m.naver.com 페이지 열기

준비단계 (TASK)

준비 완료 check

작업시작 (버튼 누르면)
- 각 브라우저에 task 한개씩 처리..
- 각 브라우저 시작시간은 random 0~500ms 대기 후 시작
- 모든 작업 끝날때까지 대기
- 모든 작업이 끝나면.. 평가하고, block 된 profile은 새로운 profile / proxy ip 로 다시 준비.
- 정상 작업 된 task 는 작업목록에서 완료처리하고 블락된 처리에 대해서는 다시 시도할 수 있도록 목록에 다시 넣는다.
- random delay (1초 이내) 후 다시 루프
