# main work flow

- Profile 갯수에 맞춰 Browser 준비

- Loop Start

  - Task List 가져오기 (n개)
  - Task 가 없으면
    - 10분간 Delay
    - continue

  - Task 병렬 처리
  - delay (대략 8초에서 15초)
  
  - 완료/실패(block)된 Task 처리
  - today stop user 처리
  - block 브라우저 처리

-Loop End
